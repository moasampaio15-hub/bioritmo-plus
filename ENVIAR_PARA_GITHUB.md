# Instruções para Enviar para GitHub

## Opção 1: Terminal (Mais rápido)

Abra o terminal na pasta `bioritmo+git` e execute:

```bash
git push origin master --force
```

Se pedir login:
- Username: moasampaio15-hub
- Password: seu token de acesso pessoal do GitHub

## Opção 2: GitHub Desktop

1. Baixe e instale: https://desktop.github.com
2. Abra o GitHub Desktop
3. File → Add local repository
4. Selecione a pasta `bioritmo+git`
5. Clique em "Publish repository"
6. Escolha "bioritmo-plus" como nome

## Opção 3: Upload Manual

1. Acesse: https://github.com/moasampaio15-hub/bioritmo-plus
2. Clique em "Add file" → "Upload files"
3. Arraste todos os arquivos da pasta `bioritmo+git`
4. Commit changes

## Dados do Repositório

- **URL:** https://github.com/moasampaio15-hub/bioritmo-plus
- **Branch:** master (ou main)
- **Arquivos:** 40 arquivos
- **Tamanho:** ~15MB

## Após enviar:

1. Verifique se todos os arquivos estão no GitHub
2. Atualize o README se necessário
3. Configure o deploy no Render
